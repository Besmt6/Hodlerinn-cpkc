Bitsy Chatbot - WordPress Embed Instructions

YES - The floating chatbot will appear on EVERY page!
Once installed, Bitsy will appear as a floating button in the bottom-right corner of ALL pages on your WordPress site.

Option 1: Using a Plugin (Easiest)

Install the "Insert Headers and Footers" plugin by WPCode, then paste this code in the Footer section:

<!-- Bitsy Chatbot Widget for Hodler Inn -->
<div id="bitsy-chat-widget"></div>

<style>
#bitsy-chat-widget {
    position: fixed;
    bottom: 20px;
    right: 20px;
    z-index: 99999;
    font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
}

#bitsy-chat-button {
    width: 60px;
    height: 60px;
    border-radius: 50%;
    background: linear-gradient(135deg, #1e3a5f 0%, #2563eb 100%);
    border: none;
    cursor: pointer;
    box-shadow: 0 4px 20px rgba(30, 58, 95, 0.4);
    display: flex;
    align-items: center;
    justify-content: center;
    transition: transform 0.3s, box-shadow 0.3s;
}

#bitsy-chat-button:hover {
    transform: scale(1.1);
    box-shadow: 0 6px 25px rgba(30, 58, 95, 0.5);
}

#bitsy-chat-button svg {
    width: 28px;
    height: 28px;
    fill: white;
}

#bitsy-chat-iframe-container {
    display: none;
    position: fixed;
    bottom: 90px;
    right: 20px;
    width: 380px;
    height: 600px;
    border-radius: 16px;
    overflow: hidden;
    box-shadow: 0 10px 40px rgba(0, 0, 0, 0.2);
    background: white;
}

#bitsy-chat-iframe-container.open {
    display: block;
}

#bitsy-chat-iframe {
    width: 100%;
    height: 100%;
    border: none;
}

@media (max-width: 480px) {
    #bitsy-chat-iframe-container {
        width: calc(100vw - 40px);
        height: calc(100vh - 120px);
        bottom: 80px;
        right: 20px;
    }
}
</style>

<script>
(function() {
    var widget = document.getElementById('bitsy-chat-widget');
    
    // Create chat button
    var button = document.createElement('button');
    button.id = 'bitsy-chat-button';
    button.innerHTML = '<svg viewBox="0 0 24 24"><path d="M20 2H4c-1.1 0-2 .9-2 2v18l4-4h14c1.1 0 2-.9 2-2V4c0-1.1-.9-2-2-2zm0 14H6l-2 2V4h16v12z"/></svg>';
    button.title = 'Chat with Bitsy';
    widget.appendChild(button);
    
    // Create iframe container
    var container = document.createElement('div');
    container.id = 'bitsy-chat-iframe-container';
    container.innerHTML = '<iframe id="bitsy-chat-iframe" src="https://cpkc.hodlerinn.com/book" allow="microphone"></iframe>';
    widget.appendChild(container);
    
    // Toggle chat
    button.addEventListener('click', function() {
        container.classList.toggle('open');
    });
    
    // Close when clicking outside
    document.addEventListener('click', function(e) {
        if (!widget.contains(e.target) && container.classList.contains('open')) {
            container.classList.remove('open');
        }
    });
})();
</script>

Option 2: Theme Footer (theme's footer.php)

Add the same code above just before the closing </body> tag in your theme's footer.php file.

Option 3: Simple Link Button

If you prefer a simple "Book Now" button on specific pages only:

<a href="https://cpkc.hodlerinn.com/book" 
   target="_blank" 
   style="display: inline-block; padding: 15px 30px; background: linear-gradient(135deg, #1e3a5f, #2563eb); color: white; text-decoration: none; border-radius: 8px; font-weight: bold;">
   Book Now with Bitsy
</a>
Important Notes:
  • The chatbot URL is: https://cpkc.hodlerinn.com/book
  • Make sure your production site is deployed before using this embed code
  • The allow="microphone" attribute enables voice input in the iframe
  • The widget is mobile-responsive and will adjust on smaller screens
Before Going Live:
Ensure the chatbot features are deployed to your production server at cpkc.hodlerinn.com first.
