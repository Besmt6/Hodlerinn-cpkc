Bitsy Chat Button - WordPress Installation Guide

Floating Chat Button Installation

This guide will help you add the Bitsy AI Booking Assistant floating button to the Hodler Inn WordPress website. The button will appear on every page, allowing visitors to easily book rooms.

Preview

💬 Chat with Bitsy

Button appears in bottom-right corner of every page

Installation Steps

1 Log in to WordPress Admin
Go to: hodlerinn.com/wp-admin
2 Navigate to Theme Editor or use a Plugin
Option A: Go to Appearance → Theme Editor → footer.php
Option B: Install plugin "Insert Headers and Footers" and add code to Footer section
3 Copy and Paste the Code Below
Add this code just before the closing </body> tag

The Code

<!-- Bitsy Chat Button - Hodler Inn --> <style> #bitsy-chat-btn { position: fixed; bottom: 20px; right: 20px; width: 60px; height: 60px; background: linear-gradient(135deg, #f59e0b, #d97706); border-radius: 50%; display: flex; align-items: center; justify-content: center; cursor: pointer; box-shadow: 0 4px 20px rgba(245,158,11,0.5); z-index: 9999; transition: transform 0.3s, box-shadow 0.3s; text-decoration: none; } #bitsy-chat-btn:hover { transform: scale(1.1); box-shadow: 0 6px 25px rgba(245,158,11,0.7); } #bitsy-chat-btn svg { width: 30px; height: 30px; fill: #000; } #bitsy-chat-btn .badge { position: absolute; top: -5px; right: -5px; background: #10b981; color: white; font-size: 10px; padding: 3px 6px; border-radius: 10px; font-weight: bold; } </style> <a id="bitsy-chat-btn" href="https://cpkc.hodlerinn.com/book" title="Chat with Bitsy - Book Your Room"> <span class="badge">Online</span> <svg viewBox="0 0 24 24"><path d="M20 2H4c-1.1 0-2 .9-2 2v18l4-4h14c1.1 0 2-.9 2-2V4c0-1.1-.9-2-2-2zm0 14H6l-2 2V4h16v12z"/></svg> </a> <!-- End Bitsy Chat Button -->
4 Save Changes
Click "Update File" or "Save" button
5 Test the Button
Visit hodlerinn.com and look for the golden chat button in the bottom-right corner

What the Button Does

✅ When visitors click the button:
  • Opens the Bitsy AI booking assistant
  • Guests can book rooms through natural conversation
  • Collects: Name, Email, Phone, Check-in/out dates, Room preference
  • Creates reservation automatically
  • Sends confirmation email to guest
  • Notifies admin via Telegram

Troubleshooting

⚠️ Button not showing?
  • Clear your browser cache (Ctrl+Shift+R)
  • Clear WordPress cache if using caching plugin
  • Make sure code is placed before </body> tag
  • Check for JavaScript errors in browser console (F12)

Customization Options

Change button position: Edit bottom: 20px; right: 20px; to adjust placement

Change button size: Edit width: 60px; height: 60px;

Change colors: Edit #f59e0b (gold) and #d97706 (dark gold)
